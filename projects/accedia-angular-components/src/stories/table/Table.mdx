# Table

The <strong>Table</strong> component provides a user-friendly way to display tabular data in web applications. It allows you to define table headers and populate rows with data efficiently. The component supports sorting and filtering functionalities to enhance data interaction.

<Canvas of={TableStories.Primary} />

<Controls />

## Functionality

<ul>
  <li>The component consists of a table structure with customizable headers and rows.</li>
  <li>You can define table headers and populate rows with data dynamically using the provided `TableRow` component.</li>
  <li>It supports sorting by columns and filtering data based on keywords.</li>
  <li>By adding the table row component as a tag `acc-table-row` on top of `tr` preserves all metadata of the `tr` base html Element which is semantically better and is more stable on the browser itself. </li>
</ul>

## Usage

To use the <strong>Table</strong> component, render it inside any template within your application via the `acc-table` selector and provide the `tableHeaders` property to specify the table headers. You can also customize the appearance by setting the `theadColor` and `theadBackgroundColor` properties.

Here is an example of how you can use the <strong>Table</strong> component in your application:

```html
<acc-table
  [tableHeaders]="yourTableHeaders"
  [data]="yourTableData"
>
  <!-- You can use the provided TableRow component for populating rows -->
  <tr acc-table-row>
    <!-- Add your columns here -->
    <td>...</td>
    <td>...</td>
    <td>...</td>
  </tr>
</acc-table>

```
