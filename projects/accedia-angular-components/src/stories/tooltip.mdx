# tooltip

The <strong>tooltip</strong> component can wrap any element to quickly show a desired tooltip inheriting the child element dimensions with content projection. Afterwards we can select our desired position of the tooltip itself. In the example we have used a button component as our `<ng-content>`. In React we refer this as a child node.
