# ProgressBar

A simple progress bar that has a progress animation that can be tweaked depending on the use case.

<Canvas of={ProgressBar.Primary} />

<Controls />

## Functionality

<ul>
  <li>Customizable minimum progress animation.</li>
  <li>Adjustable animation delay.</li>
  <li>Option to enable/disable progress animation.</li>
</ul>

## Usage

To use the ProgressBar component, simply include it in your Angular application and pass the necessary inputs.

Here is an example of how you can use the ProgressBar component in your application:

```html
<acc-progress-bar 
  [minimumProgressAnimation]="50"
  [animationDelayMs]="30"
  [withProgressAnimation]="true"
  [progress]="currentProgress">
</acc-progress-bar>
```
