# Loading spinner

The <strong>Loading spinner</strong> is a simple loading indicator used 
as a fallback component of loading or processing activities in your Angular application.

<Meta of={PaginatorStories} />

<Canvas of={PaginatorStories.Primary} />

<Controls />

## Usage

To use the <strong>Loading spinner</strong> component, simply import it into your Angular 
module and add it to your template. You can customize the size of the 
spinner by using the size input property.
